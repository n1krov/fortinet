### Ver el estado de la memoria

```sh
diagnose hardware sysinfo conserve
```


### Ver informacion de alguna interfaz
```sh
diagnose hardware deviceinfo nic <port_x>
```

```sh
diagnose sys top
```

### Matar un proceso

```sh
diagnose sys kill 11 <PID>
```


```
diagnose debug
```
